Visual Explorations

Interactive geometry of cohesive ∞‑topoi, quantum modalities, and the obstruction theorem.

What this shows (plain English)

Each blue dot is a possible state of a classical system. Drag them to see how the centre comonad collapses them to a single classical point.

Mathematical description

The line represents the Gelfand spectrum of a commutative C*-algebra. Under the centre comonad \(Q^\diamond\), the representable sheaf \(\mathbf{y}_{M_2}\) becomes empty; only commutative algebras survive. This interactive model shows that collapse: all points coalesce into the unique trivial character.

What this shows (plain English)

Adjust the spread of the points with the slider: when the spread is zero, the classical core is a single point—the system has no quantum degrees of freedom left.

Mathematical description

The slider parameter \(s\) controls the dimension of the centre. For \(s=0\), the algebra is simple and non‑commutative; the hom‑set \(\Hom(\mathbb{C}, M_n)\) is empty, so the state space collapses to a point. For \(s>0\), multiple commutative summands exist, giving a larger spectrum.

What this shows (plain English)

Pinch‑zoom and drag the diagram. It shows the failure of the diagonal map \(\Delta\) to commute with Day convolution. Toggle the button to see the collapse when the classical core is cartesian.

Mathematical description

The square encodes the naturality condition for a cloning map \(c:\mathbf{y}_{(-)}\to\mathbf{y}_{(-)}\otimes\mathbf{y}_{(-)}\). When the classical core is cartesian, the Seely isomorphism trivialises and the two sides of the square coincide.

What this shows (plain English)

This canvas lets you draw your own diagram. Use it to explore how the obstruction changes when you change the underlying site.

Mathematical description

A freehand sketch of the precomposition obstruction. The mathematical content is the same as the left diagram, but the open canvas invites the reader to experiment.

What this shows (plain English)

Rotate, zoom, and pan the 3D scene. The dark spheres are geometric manifolds; the blue columns are algebraic Hilbert spaces. Click Apply Q to see the quantum comonad project the columns to their commutative core.

Mathematical description

\( \mathbf{E} = \Sh(\mathbb{S}, J) \) with \(\mathbb{S} = \Mfd \times \Hilb_{\CP}^{\mathrm{op}}\). The spheres represent objects in \(\Mfd\); the columns represent \(\Hilb_{\CP}^{\mathrm{op}}\). \(Q^\diamond\) acts as \(i_! i^*\) with \(i: \cCAlg_{\CP}^{\mathrm{op}} \hookrightarrow \Hilb_{\CP}^{\mathrm{op}}\), collapsing each column to the classical subcategory.

What this shows (plain English)

Another view: a torus knot (representing a mixed distributive law) rotates in space. You can twist it from every angle.

Mathematical description

The torus knot parametrises the entwining map \(\psi(g\otimes\delta_x) = \delta_{gxg^{-1}}\otimes g\) for \(G = S_3\). The knot’s crossings encode the four axioms of a mixed distributive law.

What this shows (plain English)

A tesseract (4D hypercube) rotating. The coloured edges show how the distributive law \(\lambda\) maps one monad–comonad pair to the other.

Mathematical description

The 4D rotation illustrates the coherence cube of the mixed distributive law between \(T\) (group algebra monad) and \(S\) (dual coalgebra comonad). Each edge is a component of \(\lambda\).

What this shows (plain English)

A second tesseract showing the inverse distributive law. The winding edges show that the inverse exists because the entwining map is invertible.

Mathematical description

The inverse \(\lambda^{-1} = \psi^{-1}\otimes\id\) is constructed from the inverse entwining \(\psi^{-1}(\delta_y\otimes g)=g\otimes\delta_{g^{-1}yg}\). This tesseract visualises the symmetry.
